# VoltGH Store ⚡🇬🇭

Ghana's #1 Electronics & Home Appliances Online Store — built with pure HTML, CSS & JavaScript.

## 🚀 How to Run

Just open `index.html` in your browser — **no server or npm required!**

## 📁 File Structure

```
voltgh-store/
├── index.html          ← Main store (SPA — all pages in one file)
├── admin.html          ← Admin dashboard
├── css/style.css       ← All styles, dark mode, responsive
└── js/
    ├── storage.js      ← localStorage service + ImgBB upload + seed data
    ├── app.js          ← Main app logic
    └── admin.js        ← Admin dashboard logic
```
